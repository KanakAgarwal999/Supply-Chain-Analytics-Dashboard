# Power BI Dashboard

## Dashboard Pages

### 1. Overview
Provides a high-level summary of:

- Total Revenue
- Total Cost
- Products Sold
- Stock Levels
- Profit Margins

### 2. Product Insights

Analyzes:

- SKU Performance
- Product Type Revenue
- Stock Availability
- Lead Time Analysis
- Order Quantity Trends

### 3. Supplier Insights

Evaluates:

- Supplier Profitability
- Supplier Costs
- Defect Rates
- Transportation Performance

### 4. Key Insights

Summarizes business findings and recommendations.

## Tools Used

- Power BI Desktop
- DAX
- Power Query
