# 🚚 Supply Chain Analytics Dashboard

## 📌 Project Overview

This project analyzes supply chain operations using Power BI.

The dashboard provides insights into:

- Revenue performance
- Product sales
- Supplier effectiveness
- Inventory levels
- Transportation efficiency
- Profitability drivers

---

## 🎯 Business Objectives

- Monitor supply chain KPIs
- Identify top-performing products
- Evaluate supplier performance
- Track logistics costs
- Improve inventory planning
- Reduce operational inefficiencies

---

## 🛠 Tools Used

- Power BI Desktop
- Power Query
- DAX
- Microsoft Excel

## 🔍 Key Findings

### Product Analysis

- Skincare products generated the highest revenue.
- A small number of SKUs contribute significantly to total revenue.
- Manufacturing lead time impacts product availability.

### Supplier Analysis

- Supplier 3 achieved the highest average profit margin.
- Supplier performance varies significantly across cost and quality metrics.
- Some suppliers show high profitability but elevated defect rates.

### Logistics Analysis

- Carrier B contributes the highest revenue.
- Road transportation has the highest logistics cost.
- Defect rates differ across transportation modes.
