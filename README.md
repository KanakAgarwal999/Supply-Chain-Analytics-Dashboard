# Supply Chain Dataset

## Description
This dataset contains supply chain operational data including:

- Product information
- Revenue and sales metrics
- Stock levels
- Supplier details
- Manufacturing costs
- Shipping costs
- Transportation modes
- Defect rates
- Lead times

## Purpose
The dataset was used to analyze:

- Product performance
- Supplier efficiency
- Logistics operations
- Inventory management
- Profitability drivers

## Tool Used

- Microsoft Excel
- Power BI
